# Comparative Analysis of Strand, Card, and Shell Fur Techniques for Real-time Rendering

> Luke Sakal

Rendering fur has been an ongoing challenge in computer graphics, especially in real-time rendering where efficiency is important. Three techniques are common -- rendering fur as individual strands, using shells to create volumetric fur, and using hair cards. Explicit strand rendering has become more viable in real-time rendering thanks to advances in hardware. Even so, the greater efficiency that shell fur and hair cards provides can be worth the trade-off in visual quality. This project hopes to quantify the efficiency differences between these techniques under different circumstances.
